The following section will show you how to use Yahoo's OpenID Connect.

Step 1: Create Application

You'll need to go to https://developer.yahoo.com/apps/create/ and create an application.

For the purpose of this demo, use the following callback URL: 'https://zhouyaoji.github.io/openid_connect_example/index.html'

Once you've created your application, you will be given a Consumer Key and Consumer Secret. We'll be using these in the next step.
